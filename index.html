OpenClaw 能干嘛？

一句话：它是一个帮你干活的 AI 助手，不是只会聊天的机器人。

你现在用的 ChatGPT / 豆包 / Kimi

你问它问题，它回答你。仅此而已。
它不能帮你发邮件，不能帮你整理文件，不能帮你操作电脑。

用上 OpenClaw 之后

你发一条消息，它真的去做。
帮你发邮件、管日程、整理文件、搜集信息、控制浏览器，全自动。

用你熟悉的聊天工具操作

NPM 安装（最简单）

npm install -g openclaw

适合有 Node.js 环境的用户，一行命令安装完成。

  • 需要 Node.js 18+
  • 需要 2GB+ 内存

Docker 部署（推荐）

curl -fsSL https://docs.openclaw.ai/docker-setup.sh | bash

官方 Docker 一键脚本，自动配置环境和持久化存储。

  • 需要安装 Docker
  • 自动挂载 ~/.openclaw 配置
  • 最小 2GB 内存

社区一键脚本

curl -fsSL https://install.openclaw.ai | bash

社区维护的安装脚本，自动检测环境并完成安装。

本地离线运行（Ollama）

配合 Ollama 运行本地模型，完全离线，无需 API 费用。

ollama pull devstral:24b
openclaw --provider ollama --model devstral:24b
  • 需要 16GB+ 内存（运行 24B 模型）
  • 完全离线，零成本
  • 响应速度取决于硬件

常见问题

OpenClaw 安全吗？

OpenClaw 是开源的，代码完全透明。它运行在你自己的设备上，数据不会上传到第三方。但因为它能访问你的邮件、文件等，请确保你信任运行环境的安全性，不要在公共网络暴露你的 OpenClaw 实例。

我完全不懂编程，能用吗？

可以。选择 OpenClaw Cloud、AutoClaw（智谱）或阿里云一键部署等方案，不需要写任何代码。安装完成后，你只需要通过聊天软件发消息就能使用。

一个月大概要花多少钱？

取决于你的使用频率和选择的模型。轻度使用（每天几次对话）大概 $5-15/月。如果用 DeepSeek 这样的低价模型可以更便宜。用本地模型 + Oracle 免费云可以做到零成本。

OpenClaw 和 ChatGPT 有什么区别？

ChatGPT 只能聊天。OpenClaw 可以真正帮你做事——发邮件、管理文件、控制电脑、执行自动化任务。它是一个"AI 代理"，而不是"AI 聊天机器人"。

哪个版本最适合中国用户？

推荐 AutoClaw（智谱一键安装版）或 OpenClaw 中文版（社区汉化版）。它们针对国内网络环境做了优化，内置了钉钉、企业微信等国内平台支持。企业用户可以考虑 WorkBuddy（腾讯）或 CoPaw（阿里云）。
