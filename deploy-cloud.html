第三步：连接到你的服务器

买好服务器后，你需要"登录"到它上面去操作。就像远程控制一台电脑。

A

Mac 用户

打开终端（启动台搜索"终端"），输入：

# 把 123.456.789.0 替换成你的服务器 IP
ssh root@123.456.789.0

第一次连接会问你 "Are you sure you want to continue connecting?"，输入 yes 回车。

然后输入你的 root 密码（输入时屏幕不会显示字符，这是正常的），回车。

看到类似 root@server:~# 的提示符就说明连接成功了。

B

Windows 用户

Windows 10/11 自带 SSH 工具。打开 PowerShell（开始菜单搜索 "PowerShell"），输入：

# 把 123.456.789.0 替换成你的服务器 IP
ssh root@123.456.789.0

操作和 Mac 一样：输入 yes → 输入密码 → 看到服务器提示符就成功了。

如果 ssh 命令不可用：下载 PuTTY，填入 IP 地址和端口 22，点 Open，输入用户名 root 和密码即可。

第四步：安装 OpenClaw 并配置大模型

在服务器上执行几条命令装好 OpenClaw，然后把第二步拿到的 API Key 填进去。

1

安装 Docker

在服务器终端里依次输入以下命令（可以一行行复制粘贴）：

# 更新系统
apt update && apt upgrade -y

# 一键安装 Docker
curl -fsSL https://get.docker.com | sh

# 验证安装
docker --version

看到 Docker 版本号就说明安装成功。

2

安装并启动 OpenClaw

# 下载并启动 OpenClaw
docker pull openclaw/openclaw:latest

# 创建配置目录
mkdir -p ~/.openclaw

# 启动 OpenClaw
docker run -d \
  --name openclaw \
  --restart always \
  -p 18789:18789 \
  -v ~/.openclaw:/root/.openclaw \
  openclaw/openclaw:latest

等待下载完成并启动。这一步可能需要几分钟，取决于你的网络速度。

3

验证 OpenClaw 是否运行

# 查看运行状态
docker ps

如果看到 openclaw 容器状态为 Up，说明已经在运行了。

现在打开浏览器，访问：

http://你的服务器IP:18789

能看到 OpenClaw 的控制面板就说明安装成功了！

4

填入 API Key

在 OpenClaw 控制面板里：

  1. 进入 Settings（设置）页面
  2. 找到 LLM Provider（大模型提供商）
  3. 选择你注册的那个平台（如 Anthropic、DeepSeek）
  4. 粘贴你第二步拿到的 API Key
  5. 点击保存

第五步：开始使用！

恭喜！你的 AI 代理已经准备好了。

1

连接你的消息平台

在 OpenClaw 控制面板里，你可以连接各种消息平台，之后直接在这些平台上跟你的 AI 代理对话：

  • Telegram — 最推荐，配置最简单
  • 微信 — 需要额外配置，有封号风险
  • Discord — 适合游戏/社区场景
  • 飞书 / 钉钉 — 适合工作场景
2

试试给它发消息

连接好消息平台后，试着发这些消息：

"帮我查一下明天北京的天气"
"写一封请假邮件，明天有事"
"帮我总结一下这个链接的内容：xxx"
"每天早上 8 点给我发一份今日新闻摘要"
3

日常维护

基本不需要维护。以下是偶尔可能用到的命令：

# 查看 OpenClaw 运行状态
docker ps

# 查看日志（排查问题时用）
docker logs openclaw --tail 50

# 更新到最新版本
docker pull openclaw/openclaw:latest
docker stop openclaw && docker rm openclaw
docker run -d \
  --name openclaw \
  --restart always \
  -p 18789:18789 \
  -v ~/.openclaw:/root/.openclaw \
  openclaw/openclaw:latest

🎉 搞定了！
