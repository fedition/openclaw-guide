第一章

为什么 AI 最近几年这么火？

其实 AI 研究已经搞了几十年了。只是从 2022 年底开始，普通人才真正用上了。

不是突然的，是积累到了一个爆发点

科学家研究 AI 已经超过 60 年了。但在很长一段时间里，AI 只能做一些很简单的事，比如下棋、识别图片里的猫。

直到最近几年，几件事同时发生了：

  • 算力变强了——显卡（GPU）变得超级强大，能处理海量数据
  • 数据变多了——互联网上积累了几十年的文字、图片、视频，成了 AI 的"教材"
  • 方法变巧了——研究人员发现了一种叫 "Transformer" 的训练方法，效果特别好

这三件事叠在一起，就像手机从"大哥大"突然变成了智能手机——不是某一项技术的突破，是所有东西同时到位了。

ChatGPT：让普通人第一次"摸到"AI

2022 年 11 月，OpenAI 发布了 ChatGPT。这是第一次，不懂技术的普通人可以直接跟 AI 对话，而且 AI 的回答居然"像人话"。

两个月内，ChatGPT 的用户数突破了 1 亿。之后各大公司纷纷跟进：

公司产品你可能在哪里见过
OpenAIChatGPT最早火起来的
GoogleGeminiGoogle 搜索里的 AI 回答
AnthropicClaude很多人觉得写东西最好的
百度文心一言国内最早一批
字节跳动豆包抖音系的 AI 助手
阿里通义千问淘宝、钉钉里的 AI
DeepSeekDeepSeek又便宜又好用的黑马

所以，现在是什么阶段？

打个比方：现在的 AI 相当于 2010 年的智能手机。

iPhone 已经出来了（ChatGPT），各种品牌也出来了（各家大模型），但大多数人还在观望，不太知道怎么用，觉得"好像挺厉害但跟我没关系"。

但回想一下：到了 2015 年，你能想象没有手机支付、没有外卖 App 的生活吗？

AI 大概率也会走这条路。不是让你焦虑，而是说——现在了解一下，总比后面被动好。

这一章的重点
  • AI 不是突然出现的，是技术积累到了爆发点
  • ChatGPT 让普通人第一次能直接用上 AI
  • 现在的 AI 相当于 2010 年的智能手机，还在早期

一句话说清楚 OpenClaw 是什么

OpenClaw 是一个开源的 AI 代理。

上一章说了，ChatGPT、豆包这些 AI 只能聊天——你问一句它答一句。而 OpenClaw 让 AI 不止能说，还能做。

它让你通过微信、钉钉、飞书、Telegram 这些你已经在用的聊天工具，给 AI 发指令，AI 就会真的去帮你做事。比如：

第三章

常见的一些术语到底什么意思？

你不需要懂算法、不需要懂编程。但下面这几个词搞清楚了，后面看任何 AI 相关的东西都不会懵。

大模型 = AI 的"大脑"

你听到的 ChatGPT、Claude、DeepSeek，这些其实都是"大模型"——你可以理解为不同公司训练出来的"AI 大脑"。

就像手机有华为、苹果、小米，AI 大脑也有不同品牌。每个品牌的大脑有不同特点：

  • Claude——写东西、分析文档特别好
  • GPT——综合能力强，啥都能干
  • DeepSeek——又便宜又好用，写代码很强
  • 豆包/千问——中文好，国内用起来方便

记住一点就行：大模型 = AI 的大脑，不同品牌各有特长。

Token = AI 收费的"字数"

AI 不是按"次"收费的，而是按处理了多少文字来收费。这个计量单位叫 Token。

大致可以理解为：

1 个汉字 ≈ 1-2 个 Token
一篇 1000 字的文章 ≈ 1500 个 Token
1 百万 Token ≈ 一本普通小说的长度

不同的大模型价格差异很大。以处理 100 万个 Token（约一本小说）为例：

模型价格（约）类比
Claude Opus（最贵）¥100+请了个资深顾问
Claude Sonnet（主流）¥20请了个靠谱助理
GPT-4o¥15也是靠谱助理
DeepSeek V3（便宜）¥2实习生价格，但活干得不错
本地模型（免费）¥0自己家电脑跑，不花钱

日常轻度使用，一个月大概花 ¥30-100。如果用便宜模型或本地模型，可以做到接近零成本。

AI Agent = 能帮你干活的 AI

前面说了，ChatGPT、豆包这些是"问答"——你问一句它答一句。

AI Agent（AI 代理）在大脑的基础上加了"手脚"——让 AI 不只是说，还能做。OpenClaw 就是目前最火的 AI Agent。

普通 AI（聊天机器人）

你："帮我写封请假邮件"
AI："好的，这是邮件内容……（你自己复制粘贴去发）"

→
AI Agent（OpenClaw）

你："帮我写封请假邮件发给老板"
AI：（直接写好并发出去了）"已经发送完毕。"

开源 = 代码公开，免费用

"开源"的意思很简单：软件的代码是公开的，任何人都可以看、可以用、可以改。

为什么这对你很重要？

  • 免费——不用花钱买软件本身
  • 透明——代码公开，没法偷偷搜集你的数据
  • 可控——不满意可以自己改，或者用别人改好的版本
  • 不会被卡脖子——不依赖某一家公司，它倒闭了你也能继续用

Prompt = 你对 AI 说的话

你可能听过"写 prompt""prompt 工程师"之类的说法。

其实 Prompt 就是你输入给 AI 的文字。你打的每一句话、每一个指令，都是 Prompt。

写得越清楚，AI 的回答就越好。这就像你跟人交代事情一样——说得模糊就做得模糊，说得详细就做得到位。

模糊的 Prompt

"帮我写个东西"
→ AI 不知道写什么，随便给你糊一篇

→
清楚的 Prompt

"帮我写一封给客户的道歉邮件，因为发货延迟了 3 天，语气诚恳但专业"
→ AI 给你一封非常贴合的邮件

API = 软件之间的"对接口"

你可能会在部署 OpenClaw 的时候遇到"API Key"这个词。

API 就是两个软件之间沟通用的接口。OpenClaw 需要通过 API 去调用大模型的能力。而 API Key 就是你的"通行证"——大模型公司给你一串密码，OpenClaw 拿着这个密码去调用它的服务。

就像你家的 WiFi 密码——有了密码才能上网，有了 API Key，OpenClaw 才能调动 AI 大脑。

这一章的重点
  • 大模型 = AI 的大脑，不同品牌各有特长
  • Token = AI 收费的计量单位，约等于字数
  • AI Agent = 能真正帮你干活的 AI，不只是聊天
  • 开源 = 免费、透明、可控
  • Prompt = 你对 AI 说的话，说得越清楚效果越好
  • API / API Key = 软件之间的对接口 / 你的通行证

我应不应该现在用这些东西？

看到这里，你可能在想：这东西到底跟我有没有关系？我现在就得用吗？

先说结论

不是所有人都需要现在就用。但了解一下，绝对不亏。

这就像 2012 年的移动支付——当时很多人觉得"我又不是年轻人，用现金挺好的"。结果几年之后，不会扫码付款的人在菜市场都不方便了。

AI 不一定会走这么快，但方向是确定的。

这几类人，建议现在就试试

你是这种情况为什么建议你试
每天有大量重复性工作（整理表格、回邮件、写报告）OpenClaw 能帮你自动化这些事，省出来的时间是实打实的
工作中需要处理大量文本（法务、财务、行政）AI 最擅长的就是文本处理，效率提升特别明显
对新技术好奇，愿意花半天时间折腾现在上手门槛已经很低，半天就能跑起来
团队里有人在用，你想跟上节奏先了解原理，再看别人怎么用，然后自己试

这几类人，可以先观望

  • 工作完全是体力活或手工活——AI 目前主要帮的是"脑力活"和"电脑上的活"
  • 对电脑操作本身就不熟练——可以先从最简单的聊天 AI（豆包、ChatGPT）开始用，等熟悉了再考虑 OpenClaw

不管你用不用，记住两件事

第一：不要焦虑

AI 不会明天就替代你。它目前还有很多做不好的事（比如需要创造力的工作、需要人际关系判断的工作）。网上那些"不学 AI 就被淘汰"的说法，大部分是在卖课。

你不需要成为专家，但知道 AI 能做什么、不能做什么，会帮你做出更好的判断。就像你不需要会修手机，但你得会用手机。
