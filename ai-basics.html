第一章

为什么 AI 最近几年这么火？

一句话：因为 2022 年底出了个 ChatGPT，让所有人第一次感受到"AI 居然能像人一样说话"。

以前的 AI：只会做选择题和填空题

AI 研究其实已经搞了几十年了。但以前的 AI 本质上是根据设定好的问答集来回答问题——有人提前写好了"用户问 A 就回答 B"，没写到的问题，它就不会回答。

你可能遇到过那种客服机器人，问什么都答非所问，稍微换个说法它就不懂了。那就是旧时代的 AI。

简单来说：以前的 AI 只会做选择题和填空题。

ChatGPT 开始：AI 会做应用题、写作文了

2022 年 11 月，OpenAI 发布了 ChatGPT。它和以前的 AI 完全不同——

ChatGPT 不是根据设定好的问题集来回答，而是通过学习互联网上的海量信息，训练出了一个类似大学生一样的通才。它能够自己组织语言，自己生成句子，真正像一个人一样和你对话。

简单来说：从 ChatGPT 开始，AI 会做应用题了，能写作文了。

以前的 AI

只会做选择题和填空题
提前设定好答案，没设定的就不会
换个说法就听不懂

→
ChatGPT 之后的 AI

会做应用题，能写作文
学习了互联网海量信息，自己生成回答
像一个通才大学生一样和你对话

然后所有公司都跟上了

ChatGPT 火了之后，全世界的科技公司都开始做自己的 AI：

所以你看到的豆包、元宝、千问这些，本质上都是 ChatGPT 引发的这波浪潮的产物。

这一章的重点
  • 2022 年 11 月 ChatGPT 发布，引发了整个 AI 浪潮
  • 以前的 AI 只会做选择题——根据设定好的答案回答
  • 现在的 AI 会做应用题——像个通才大学生，能自己组织语言和你对话
  • 豆包、千问这些都是这波浪潮的产物
第二章

了解一下 AI 现在常见的黑话

你不需要懂算法、不需要懂编程。但下面这几个词搞清楚了，后面看任何 AI 相关的东西都不会懵。

为了让你一次性搞懂所有概念，我们用一个连贯的比喻来串起来：

🎓 一个比喻贯穿全章

想象你雇了一个刚毕业的研究生。他学习能力很强，什么都懂一点，能上网查资料，也能跟你正常对话沟通。

AI 的世界，就是围绕这个"研究生"展开的。

先看全景：所有概念怎么拼在一起

下面这张图展示了所有术语之间的关系。看不懂没关系，后面一个一个讲。

▲ 所有术语之间的关系全景图

🎓 大模型 = 一个刚毕业的研究生

你可以把 AI 大模型想象成一个刚毕业的研究生——他学习能力很强，读过海量的书、论文、网页，什么都懂一点。他能自己上网查资料，也能跟你正常对话，基本可以按照你的要求给你回答问题。

但就像研究生有名校和普校之分一样，不同"大模型"也有好坏之别：

重要的是：你随时可以换人。用了 Claude 觉得贵，可以换 DeepSeek；想省钱，就用本地模型。OpenClaw 不绑定任何一个研究生。

💰 Token = 给研究生发的工资

研究生不是白干的，要发工资。但不是月薪，而是按量计费——他帮你处理了多少文字，你就付多少钱。

这个计量单位叫 Token，大致等于文字数量：

1 个汉字 ≈ 1-2 个 Token
一篇 1000 字的文章 ≈ 1500 个 Token
1 百万 Token ≈ 一本普通小说的长度

不同研究生的"工资"差距很大：

日常轻度使用，一个月大概花 ¥30-100。简单任务用便宜的研究生，重要任务换贵的来做——混合使用，最省钱。

🛠 Skills = 研究生的技能

这位 AI 研究生是个通才——什么都会一点，什么都可以学。

当你有具体明确的问题需要他解决的时候，就可以让他去学习特定的技能（Skills），这样他能把你想要的事情做得更好、更专业。

OpenClaw 社区有大量现成的 Skills 可以安装，就像给研究生报培训班——技能越多，能帮你干的事就越多。

💬 豆包/千问 vs 🖥️ OpenClaw：差别在哪？

理解了"研究生"之后，这个区别就很好懂了：

豆包、元宝、千问这些，就是一个只能上网、只能和你聊天的研究生。你问他问题他能答，但让他帮你发个邮件？他做不到。

OpenClaw 呢，就是你给这个研究生一台电脑。你可以把你自己的电脑给他用，也可以另外租一台专门给他。这样他就可以像真人一样操作电脑，帮你去完成电脑上的各种操作。

🔑 API Key = 研究生的「上岗证」

你选好了要雇哪个研究生（比如 Claude），就得去他所在的"学校"（Anthropic 公司）办一张上岗证。有了这张证，研究生才能正式来给你干活。

这张上岗证就是 API Key——一串密码，OpenClaw 拿着它去调用研究生的能力，同时通过它来计费发工资。

部署 OpenClaw 的时候，只需要去对应公司网站注册 → 获取 API Key → 填进去，研究生就上岗了。

💬 Prompt = 你交代给研究生的话

你可能听过"写 Prompt""Prompt 工程师"之类的说法。其实就是你给研究生交代任务时说的话。

就像你跟真人交代工作一样——说得越清楚，他做得越好：

模糊的交代（差 Prompt）

"帮我写个东西"
→ 研究生不知道写什么，随便交了一篇

→
清楚的交代（好 Prompt）

"帮我写一封给客户的道歉邮件，因为发货延迟了 3 天，语气诚恳但专业"
→ 研究生交出一封完美的邮件

📖 开源 = 这个研究生的"培养方案"是公开的

OpenClaw 是开源的，意思是它的全部代码（相当于研究生的"培养方案"和"工作手册"）都是公开的，任何人都可以看、可以用、可以改。

这意味着：

  • 免费——OpenClaw 本身不要钱（研究生工资另算）
  • 透明——代码公开，没法偷偷搜集你的数据
  • 可控——不满意可以自己改，或者用别人改好的版本
  • 不会被卡脖子——不依赖某一家公司

串起来再看一遍

这一章的重点（用研究生比喻记住）
  • 大模型 = 一个刚毕业的研究生（Claude / GPT / DeepSeek，各有特长）
  • Token = 给研究生发的工资（干多少活付多少钱）
  • Skills = 研究生的技能（通才，什么都能学，学了就做得更好）
  • 豆包/千问 = 只给了研究生一个聊天窗口（能问答，不能操作电脑）
  • OpenClaw = 给研究生一台电脑（能像真人一样帮你操作电脑做事）
  • API Key = 研究生的上岗证（去学校注册拿到，才能正式干活）
  • Prompt = 你交代给研究生的话（说得越清楚，活干得越好）
  • 开源 = 培养方案公开（免费、透明、可控）
第三章

为什么 OpenClaw 忽然火了？

GitHub 上 24 万星标，几万个分支版本，中国大厂纷纷推出自己的版本。凭什么？

一句话说清楚 OpenClaw 是什么

OpenClaw 是一个开源的 AI 代理。

上一章说了，ChatGPT、豆包这些 AI 只能聊天——你问一句它答一句。而 OpenClaw 让 AI 不止能说，还能做。

它让你通过微信、钉钉、飞书、Telegram 这些你已经在用的聊天工具，给 AI 发指令，AI 就会真的去帮你做事。比如：

  • "帮我查一下明天的天气，整理成消息发到群里"
  • "把桌面上的发票照片识别出来，整理成 Excel"
  • "每天早上 8 点给我发一份今日待办"
  • "帮我回复这封邮件，语气正式一点"

它和 ChatGPT 到底有什么不同？

ChatGPT / 豆包 / 千问

只能聊天。你问它答。
不能操作你的电脑、不能发邮件、不能整理文件。
所有事还得你自己来。

→
OpenClaw

能真正做事。你说它做。
能发邮件、整理文件、搜信息、控制浏览器。
你只需要确认结果。

它凭什么这么火？

说白了，就是四个字：免费、好用。展开说：

  1. 免费开源——软件本身不要钱，代码公开透明
  2. 大脑自己选——Claude、GPT、DeepSeek 都行，贵的便宜的随你
  3. 平台随意——微信、钉钉、飞书、Telegram、Discord 都支持
  4. 数据在你手里——运行在你自己的设备上，隐私可控
  5. 生态丰富——有各种变种版本，中国大厂也出了定制版

我应不应该现在用这些东西？

看到这里，你可能在想：这东西到底跟我有没有关系？我现在就得用吗？

先说结论

不是所有人都需要现在就用。但了解一下，绝对不亏。

这就像 2012 年的移动支付——当时很多人觉得"我又不是年轻人，用现金挺好的"。结果几年之后，不会扫码付款的人在菜市场都不方便了。

AI 不一定会走这么快，但方向是确定的。

这几类人，建议现在就试试

你是这种情况为什么建议你试
每天有大量重复性工作（整理表格、回邮件、写报告）OpenClaw 能帮你自动化这些事，省出来的时间是实打实的
工作中需要处理大量文本（法务、财务、行政）AI 最擅长的就是文本处理，效率提升特别明显
对新技术好奇，愿意花半天时间折腾现在上手门槛已经很低，半天就能跑起来
团队里有人在用，你想跟上节奏先了解原理，再看别人怎么用，然后自己试

这几类人，可以先观望

  • 工作完全是体力活或手工活——AI 目前主要帮的是"脑力活"和"电脑上的活"
  • 对电脑操作本身就不熟练——可以先从最简单的聊天 AI（豆包、ChatGPT）开始用，等熟悉了再考虑 OpenClaw
  • 担心数据安全，所在行业有严格合规要求——可以等企业级版本（NemoClaw 等）更成熟了再用

不管你用不用，记住两件事

第一：不要焦虑

AI 不会明天就替代你。它目前还有很多做不好的事（比如需要创造力的工作、需要人际关系判断的工作）。网上那些"不学 AI 就被淘汰"的说法，大部分是在卖课。

第二：保持了解

你不需要成为专家，但知道 AI 能做什么、不能做什么，会帮你做出更好的判断。就像你不需要会修手机，但你得会用手机。
