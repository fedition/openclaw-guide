本地安装教程

在你的电脑上安装 OpenClaw，跟着步骤走就行。

1

安装 Node.js

OpenClaw 需要 Node.js 22 以上版本。如果你不确定有没有装过，打开终端（在启动台搜索"终端"），输入：

node -v

如果显示 v22.x.x 或更高，跳到第 2 步。否则：

  1. 打开 nodejs.org
  2. 下载左边的 LTS 版本（推荐）
  3. 双击安装包，一路下一步完成安装
  4. 重新打开终端，再次输入 node -v 确认版本
2

一键安装 OpenClaw

在终端里输入以下命令：

curl -fsSL https://openclaw.ai/install.sh | bash

这个脚本会自动完成安装，并启动引导向导。

不放心直接运行脚本？ 可以先下载查看内容： 
curl -O https://openclaw.ai/install.sh
cat install.sh    # 查看脚本内容
bash install.sh   # 确认没问题后再运行

或者用 npm 安装：

npm install -g openclaw@latest
openclaw onboard --install-daemon
3

获取 AI 大模型的 API Key

OpenClaw 本身只是一个工具，它需要连接 AI 大模型才能"思考"。你需要去大模型公司的网站注册并获取一个 API Key（相当于你的账号密码）。

推荐选择：

  • Claude（Anthropic）— 效果好，去 console.anthropic.com 注册，在 Settings → API Keys 创建
  • DeepSeek — 便宜，去 platform.deepseek.com 注册获取
  • 不想花钱？ 安装 Ollama，在本地运行免费模型（需要电脑内存 16GB 以上）
4

配置 API Key

安装向导会引导你填入 API Key。如果跳过了，可以手动设置：

# 以 Anthropic 为例，把下面的 sk-ant-xxx 替换成你自己的 Key
echo 'export ANTHROPIC_API_KEY="sk-ant-xxx"' >> ~/.zshrc
source ~/.zshrc

然后运行引导向导完成剩余配置：

openclaw onboard
5

验证安装

在终端输入：

openclaw --version
openclaw doctor

如果显示版本号并且 doctor 没有报错，说明安装成功！

现在你可以连接微信、Telegram、Discord 等平台，给你的 AI 代理发消息了。

Apple Silicon 用户注意：如果你用的是 M1/M2/M3/M4 芯片的 Mac，安装时一般不会有问题。如果遇到报错，可以在终端输入 node -p "process.arch"，确认显示的是 arm64。
1

安装 Docker Desktop（推荐方式）

Windows 上最简单的方式是用 Docker。不用装各种依赖，一步搞定。

  1. 打开 Docker Desktop 官网
  2. 下载 Windows 版本并安装
  3. 安装过程中勾选 "Use WSL 2 instead of Hyper-V"
  4. 安装完成后重启电脑
  5. 打开 Docker Desktop，等待它启动完成（任务栏图标变为稳定状态）
电脑要求：Windows 10 版本 2004 及以上，或 Windows 11。内存至少 8GB（推荐 16GB）。
2

用 Docker 安装 OpenClaw

打开 PowerShell（在开始菜单搜索 "PowerShell"），输入：

docker pull openclaw/openclaw:latest
docker-compose up -d

等待下载完成后，OpenClaw 就运行起来了。

或者用完整的安装脚本：

git clone https://github.com/openclaw/openclaw.git
cd openclaw
docker-compose up -d
3

获取 AI 大模型的 API Key

跟 Mac 一样，你需要去大模型公司注册获取 API Key：

  • Claude（Anthropic）— 效果好，去 console.anthropic.com 注册
  • DeepSeek — 便宜，去 platform.deepseek.com 注册
  • 不想花钱？ 安装 Ollama 运行本地免费模型（需要 16GB+ 内存）
4

打开控制面板，填入 API Key

Docker 安装完成后，打开浏览器访问：

http://127.0.0.1:18789/

在 Settings 页面粘贴你的 API Key 即可。

5

开始使用

配置完成后，你可以在控制面板里连接 Telegram、Discord 等消息平台，然后给你的 AI 代理发消息试试。

每次开机后，确保 Docker Desktop 在运行，OpenClaw 就会自动启动。

另一种方式：WSL2 + 命令行安装
如果你熟悉 Linux 命令行，也可以先安装 WSL2，然后在 WSL2 终端里像 Mac 一样用 curl 或 npm 安装。方法跟上面 macOS 部分完全一样。

常见问题

Q: 输入 openclaw 提示"命令未找到"？

关闭终端重新打开试试。如果还不行，检查 Node.js 是否安装成功（node -v），然后重新运行 npm install -g openclaw@latest。

Q: Docker 启动后显示内存不足？

打开 Docker Desktop → Settings → Resources，把内存分配调到至少 2GB。

Q: API Key 填了但还是不能用？

确认 Key 没有多余的空格，且账户余额充足。可以去对应平台的控制台检查。

Q: 想换一个 AI 模型怎么办？

运行 openclaw onboard 重新进入配置向导，或在控制面板的 Settings 里修改。
