推荐
☁️

安装到云服务器

把程序装在云端，随时随地都能用。

  • 更安全 — 程序运行在远程服务器，不会影响你的电脑，不用担心数据丢失或系统异常
  • 随时可用 — 24 小时运行，手机、平板、任何设备都能操作
  • 不占电脑资源 — 不吃你电脑的内存和 CPU，该干嘛干嘛
  • 适合长期使用 — 服务器一直在线，AI 代理可以持续工作

需要注意：

  • 需要额外购买云服务器（约 ¥30~100/月）
  • 需要购买 AI 大模型 API 费用（按使用量计费）
  • 首次配置需要 10~30 分钟
云服务器部署教程 →

推荐使用阿里云、腾讯云，或 Oracle 免费服务器

💻

安装到自己电脑上

直接装在你的 Windows 或 Mac 上。

  • 不需要买服务器 — 用你现有的电脑就行，只需要付大模型费用
  • 数据完全本地 — 所有数据都在你的电脑上，最大程度保护隐私
  • 上手快 — 下载安装，打开就能用

需要注意：

  • 需要购买 AI 大模型 API 费用（按使用量计费）
  • 电脑关机后 AI 代理就停了，无法 24 小时工作
  • 占用电脑内存和 CPU，可能影响其他工作
  • OpenClaw 有权限操作你的电脑文件，如果配置不当可能误删文件或造成其他问题
  • 只能在这台电脑上用，手机上没法操作
本地安装教程 →

需要 Windows 10+ 或 macOS 12+，内存 8GB 以上
